

● Here's a summary of the project and its bin/ scripts:

Project: gjvc-python-bootstrap

A Python 3.13 project template designed to quickly bootstrap new ideas. Per the README: "Intended to benefit the first 10 hours and days, not the first 10 months or years."

---
bin/ Scripts

Scripts follow a clear naming convention: app-* (entry points), dev-* (lifecycle), venv-* (plumbing).

Notable Patterns

- Canonical path resolution — every script uses the same 4-line realpath/BASH_SOURCE[0] idiom for location-independence
- Strict bash — set -o errexit -o nounset -o pipefail everywhere
- Environment hygiene — app-main starts from a blank env; venv-bin blocks user site-packages
- Three-tier requirements — bootstrap (minimal) → root (pinned) → deptree (full tree) → pylock.toml (hash-locked)
- CLI/Shell symmetry — the interactive cmd2 shell commands mirror the --flags on the CLI
